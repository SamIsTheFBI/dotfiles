What are dotfiles?

Dotfiles are plain text configuration files on Unix like systems like Linux, BSD and MacOS. These files store your settings of almost all applications, service and tool running on your system. These files control the looks and behavior of applications. Dotfiles are mainly used to backup, restore and sync your current settings. These can also be used for sharing your configs for the rest of the world to try out.

My dotfiles will make your PC run faster!

My Linux setup runs on a potato laptop but I needed it to be fast. With minimal programs like suckless' dynamic window manager (dwm), I have succeeded in making my 4GB RAM laptop run faster. This window manager is very lightweight and takes as little as 500MB of RAM.

You're gonna enjoy terminals with these configs!

My workflow includes spending a lot of time on terminals; that's where I code, run and deploy! My dotfiles include configuration files for all my terminal needs. With the power of pywal and bash scripting, my setup adapts the colorscheme from images that you set as wallpaper.
